**4\. Ответы на контрольные вопросы**

**Вопрос 1. Какие режимы работы есть в mc? Охарактеризуйте их.**

Ответ: В Midnight Commander существуют следующие основные режимы отображения панелей:

- **Стандартный (список файлов)** — на панели отображается список файлов и каталогов текущего каталога.
- **Информация** — на панели выводятся подробные сведения о выделенном файле (имя, размер, права, владелец, группа, время изменения, информация о файловой системе).
- **Дерево** — на панели отображается иерархическая структура каталогов файловой системы.
- **Отключённые панели** — панели временно скрываются, остаётся только командная строка (Ctrl+o).

**Вопрос 2. Какие операции с файлами можно выполнить как с помощью команд shell, так и с помощью меню (комбинаций клавиш) mc? Приведите несколько примеров.**

Ответ:

| Операция | Команда shell | Комбинация клавиш mc |
| --- | --- | --- |
| Копирование файла | cp source dest | F5  |
| Перемещение/переименование | mv source dest | F6  |
| Удаление файла | rm file | F8  |
| Создание каталога | mkdir dir | F7  |
| Просмотр файла | cat file, less file | F3  |
| Редактирование файла | nano file, vim file | F4  |
| Изменение прав доступа | chmod mode file | Ctrl+x o → Права |

**Вопрос 3. Опишите структуру меню левой (или правой) панели mc, дайте характеристику командам.**

Ответ: Меню **Левая панель** (аналогично для правой) содержит:

- **Список** — выбор формата отображения файлов (краткий, полный, расширенный и др.).
- **Сортировка** — сортировка по имени, времени изменения, размеру, без сортировки.
- **Фильтр** — возможность задать маску для отображения только определённых файлов.
- **Режим** — переключение между режимами: список, информация, дерево.
- **Перечитать** — принудительное обновление содержимого панели.

**Вопрос 4. Опишите структуру меню Файл mc, дайте характеристику командам.**

Ответ: Меню **Файл** содержит:

- **Просмотр** (F3) — просмотр содержимого файла (только чтение).
- **Редактирование** (F4) — открытие файла во встроенном редакторе.
- **Копирование** (F5) — копирование файлов/каталогов.
- **Переименование/Перемещение** (F6) — перемещение или переименование.
- **Создание каталога** (F7) — создание нового каталога.
- **Удалить** (F8) — удаление файлов/каталогов.
- **Жёсткая ссылка** (Ctrl+x l) — создание жёсткой ссылки.
- **Символическая ссылка** (Ctrl+x s) — создание символической ссылки.
- **Владелец/группа** (Ctrl+x o) — изменение владельца и группы.
- **Права (расширенные)** — изменение прав доступа.
- **Выход** (F10) — завершение работы mc.

**Вопрос 5. Опишите структуру меню Команда mc, дайте характеристику командам.**

Ответ: Меню **Команда** содержит:

- **Дерево каталогов** — отображение структуры каталогов.
- **Поиск файла** — поиск файлов по имени и содержимому.
- **Переставить панели** — меняет местами левую и правую панели (Ctrl+u).
- **Сравнить каталоги** (Ctrl+x d) — сравнение содержимого двух каталогов.
- **Размеры каталогов** — корректное отображение размеров каталогов.
- **История командной строки** — список ранее выполненных команд.
- **Каталоги быстрого доступа** (Ctrl+\\) — быстрый переход в избранные каталоги.
- **Восстановление файлов** — восстановление файлов на ext2/ext3.
- **Редактировать файл расширений** — настройка действий для файлов по расширению.
- **Редактировать файл меню** — редактирование пользовательского меню (F2).
- **Редактировать файл расцветки имён** — настройка цветов для разных типов файлов.

**Вопрос 6. Опишите структуру меню Настройки mc, дайте характеристику командам.**

Ответ: Меню **Настройки** содержит:

- **Конфигурация** — основные настройки работы панелей (показывать скрытые файлы, подтверждения и т.д.).
- **Внешний вид** — настройка внешнего вида интерфейса (Full screen, Double Width, цветовые схемы).
- **Подтверждения** — настройка запросов подтверждения при выполнении опасных операций (удаление, перезапись и т.д.).
- **Настройки панелей** — детальная настройка отображения каждой панели.
- **Настройки редактора** — настройки встроенного редактора (табуляция, подсветка синтаксиса и т.д.).
- **Сохранение настроек** — сохранение текущей конфигурации.

**Вопрос 7. Назовите и дайте характеристику встроенным командам mc.**

Ответ: Встроенные команды mc — это действия, привязанные к функциональным клавишам F1–F10:

- **F1** — помощь (справочная система).
- **F2** — пользовательское меню (настраивается через файл ~/.mc/menu).
- **F3** — просмотр файла.
- **F4** — редактирование файла.
- **F5** — копирование файла/каталога.
- **F6** — перемещение/переименование.
- **F7** — создание каталога.
- **F8** — удаление файла/каталога.
- **F9** — активация верхнего меню.
- **F10** — выход из mc.

**Вопрос 8. Назовите и дайте характеристику командам встроенного редактора mc.**

Ответ: Основные команды встроенного редактора mc:

- **Ctrl+y** — удалить текущую строку.
- **Ctrl+u** — отменить последнее действие.
- **Ctrl+Ins** — копировать выделенный фрагмент в буфер.
- **Shift+Del** — вырезать выделенный фрагмент в буфер.
- **Shift+Ins** — вставить из буфера.
- **Ctrl+Home** — перейти в начало файла.
- **Ctrl+End** — перейти в конец файла.
- **F2** — сохранить файл.
- **F4** — поиск (или переход к следующему вхождению).
- **F7** — поиск и замена.
- **F10** — выход из редактора (с запросом сохранения).

**Вопрос 9. Дайте характеристику средствам mc, которые позволяют создавать меню, определяемые пользователем.**

Ответ: Пользовательское меню в mc создаётся путём редактирования файла ~/.mc/menu. Это меню вызывается по клавише **F2**. Формат файла:

- Строки, начинающиеся с + или - (заголовки разделов).
- Строки, начинающиеся с буквы или цифры (команды).
- Каждая команда имеет вид: "Описание" → выполняемая команда.

Пример:

text

\+ Общее меню

a Открыть в nano nano %f

c Скомпилировать C gcc %f -o %f.out

Где %f — текущий файл, %d — текущий каталог. Это позволяет пользователю добавлять собственные действия для файлов.

**Вопрос 10. Дайте характеристику средствам mc, которые позволяют выполнять действия, определяемые пользователем, над текущим файлом.**

Ответ: Для выполнения пользовательских действий над текущим файлом используются:

1.  **Пользовательское меню (F2)** — позволяет запускать произвольные команды (скрипты, компиляторы, просмотрщики) для выделенного файла. Команды задаются в ~/.mc/menu.
2.  **Файл расширений** (~/.mc/ext) — позволяет привязать действия к типам файлов по расширению. Например, при нажатии Enter на файле .jpg может автоматически запускаться просмотрщик изображений.
3.  **Команда «Редактировать файл расцветки имён»** — позволяет задать цветовое выделение файлов по маске имени (например, все \*.sh выделять зелёным).
